IT Systems & Network Engineering Internship

During my internship, I took on significant responsibilities in managing and securing corporate IT infrastructures. My work focused on creating scalable network solutions and ensuring the high availability of server resources through advanced configuration and troubleshooting. This experience provided me with a deep technical understanding of how hardware and software integrate to support large-scale business operations.

Core Technical Competencies & Projects

  • Network Infrastructure & Management: I specialized in the configuration of MikroTik and Ruijie networking hardware, where I implemented VLANs and managed Access Points to ensure seamless connectivity across the facility.
  • Server Administration & Virtualization: I gained hands-on experience in setting up and managing virtual environments using VMware ESXi, including the deployment of virtual machines and the management of Active Directory for user authentication and permissions.
  • Security & Firewall Configuration: I actively participated in defining firewall rules and security protocols to protect the internal network from external threats and unauthorized access.
  • Database & Web Services: My role included the installation and maintenance of SQL databases, along with the deployment of web applications using IIS (Internet Information Services).
  • Hardware Diagnostics & Maintenance: I performed advanced troubleshooting on server and workstation hardware, including component upgrades, BIOS configurations, and diagnosing motherboard-level failures.

This internship allowed me to master professional tools and methodologies, from network mapping to complex system recovery. Working in a fast-paced IT environment sharpened my ability to diagnose system bottlenecks and implement long-term technical solutions.
