Autonomous Indoor Navigating Robot

PLUG IN

Lıdar Mapping & ROS2

This project focuses on the design and implementation of a holonomic autonomous mobile robot optimized for indoor environments using the ROS 2 (Jazzy) framework. The system architecture is built on a hybrid control model where high-level decision-making and sensor processing are handled by a Raspberry Pi 5, while low-level actuator synchronization and real-time tasks are managed by an Arduino Mega 2560. The robot features a triangular chassis equipped with three omni-directional wheels, providing full mobility in any direction and enabling precise maneuvering in confined spaces, such as narrow corridors as small as 60 cm..

A significant engineering challenge addressed during development was the compensation for hardware limitations, specifically a faulty wheel encoder and mechanical drift. To overcome this without increasing costs, I implemented a custom virtual encoder algorithm that estimates motion using motor commands and physical constraints, achieving a deviation rate of less than 2%. For environmental perception, an RPLIDAR A2M8 sensor provides 360-degree real-time distance measurements. Rather than relying on resource-heavy standard SLAM toolboxes, I developed a lightweight mapping solution based on Bresenham’s Line Algorithm, which efficiently transforms LIDAR data into a 2D occupancy grid map while maintaining high performance on embedded hardware.

The navigation pipeline utilizes a custom-designed dynamic path planning and obstacle avoidance strategy. This system analyzes environmental data across four distinct zones—front, rear, left, and right—to make instantaneous trajectory adjustments. By employing a rule-based reaction matrix, the robot can detect and bypass both static and dynamic obstacles, such as furniture or moving objects, with a mapping accuracy of approximately 95% in open areas. This project demonstrates that sophisticated autonomous functionalities, including real-time mapping and collision-free navigation, can be successfully achieved using cost-effective hardware through software optimization and robust modular integration.
