LDR-Based Solar Tracking System

This project follows the core logic illustrated in the accompanying image to achieve automated orientation. It involves the design and implementation of a light-tracking system that automatically aligns a DC motor with the strongest light source. By utilizing Light Dependent Resistors (LDRs) as sensors, the system creates a feedback loop that determines the direction of movement based on differential light intensity. The core of the project focuses on hardware-level direction control without the need for complex microcontrollers in its most basic form.

Working Principle & Circuit Architecture

The system operates by comparing the resistance values of two LDRs placed on opposite sides of the tracker. The balanced state is achieved when both sensors receive equal light. When an imbalance occurs, the following process is triggered:

  • Dual-Sensor Comparison: LDRs act as variable resistors in a voltage divider circuit, where light intensity is converted into a voltage signal.
  • Diode-Based H-Bridge Implementation: To control the DC motor's direction (clockwise and counter-clockwise), a custom H-Bridge circuit was designed. By using a specific arrangement of diodes and transistors (or switches), the circuit reverses the polarity of the voltage applied to the motor based on which LDR is triggered.
  • Bidirectional Control: This bridge configuration allows the motor to rotate towards the light source until the LDRs reach an equilibrium, effectively "tracking" the light in real-time.
